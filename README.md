# Smart Lender

Machine learning-powered web application to predict whether a loan application will be approved or rejected using the Gradient Boosting Classifier (GBC).

# Folder Structure

**6-8** — Testing, Documentation and Demonstration Deliverables

**docs** — Architecture, Project Flow and Supporting Documents

**Gradient Boosting Classifier (GBC)**

# How to Run

```bash
cd "5.Project Development Phase"

pip install -r requirements.txt
```
